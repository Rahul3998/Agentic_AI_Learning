# 🚀 CrewAI + Ollama (Gemma) – Local Execution Demo

This project demonstrates how to run a **CrewAI agent locally** using **Ollama (Gemma model)** and successfully execute a task end-to-end.

---

## 📌 Objective

To validate that:
- CrewAI can run locally
- Ollama (Gemma) integrates successfully
- Agent → Task → Crew workflow executes without errors

---

## 🛠️ Tech Stack

- CrewAI
- Ollama
- Gemma (3:4B model)
- Python (venv)
