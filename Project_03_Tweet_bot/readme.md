# Human-Like Tweet Prompt

This prompt is designed to generate tweets that feel natural, casual, and written by a real person—not overly polished or AI-generated.

## Purpose

Most AI-generated tweets sound too formal, promotional, or “perfect.” This prompt helps produce tweets that are:

* Relatable
* Conversational
* Slightly imperfect (like real human posts)

## Features

* Keeps tweets within the 280-character limit
* Encourages simple and natural language
* Avoids generic or overly hyped phrases
* Limits emoji and hashtag usage
* Allows light personal or conversational tone

## Usage

Call the `tweet_prompt()` function and provide your topic. The model will return a single tweet based on that topic.

## Output Rules

* Only one tweet is generated
* No explanations or extra text
* Tone remains casual and human-like

## Example

**Input:**
Topic: "Wishing Rahul 25th birthday!"

**Output:**
Happy 25th birthday Rahul! 🎉 Hope you get a proper break today and eat more cake than you should 😄

## Best Use Cases

* Personal messages
* Birthday wishes
* Casual social media content
* Relatable, everyday tweets
